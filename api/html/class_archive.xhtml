bool Archive::ShouldCompress ( const string &  path,
const string &  mimeType,
size_t  size 
) const
virtual

Determines whether a given file ought to be compressed when stored in the archive.

The default implementation looks at the mediaType to check for pre-compressed image, audio, or video data, for which it returns false. It will also return false for any file below 1024KB in size. For all other input, it returns true.

Parameters
pathThe path of an item. Does not need to exist in the archive.
mimeTypeThe MIME type or media-type of the data to be stored.
sizeThe number of bytes expected to be written.
Returns
Returns true if the data is considered appropriate to compress, false if the archive implementor believes compression is unnecessary.
