template<class _Rep _DEFAULT_, class _Period _DEFAULT_>
ExitReason RunLoop::Run ( bool  returnAfterSourceHandled,
std::chrono::duration< _Rep, _Period >  timeout 
)
inline

Run the RunLoop, either indefinitely, for a specific duration, and/or until an event occurs.

Parameters
returnAfterSourceHandledReturn from this method after a single EventSource has fired. Note that this only applies to EventSources– Timers will not cause this method to return.
timeoutThe maximum amount of time to run the run loop as a result of this invocation. If 0 or less, this will poll the RunLoop exactly once, firing any pending timers and a miximum of one event source.
Returns
Returns a value defining the reason that the method returned.
