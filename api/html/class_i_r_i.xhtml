Detailed Description

The IRI class encapsulates all URL and URN storage in Readium.

The EPUB 3 specification calls for IRIs rather than URIs (i.e. Unicode characters are allowed and should not be implicitly encoded) in matching properties and other identifiers. This class provides URN support internally, URL support through Google's GURL library, and Unicode IRI support is wrapped around GURL.

Definition at line 203 of file iri.cpp.

string IRI::URIString ( ) const

Obtains a valid ASCII URL representation of this IRI.

Percent-encodes all URL-reserved characters and all non-ASCII characters outside the hostname using those characters' UTF-8 representations. Uses IDN algorithm to obtain an ASCII representation of any Unicode hostname.

Returns
An ASCII URL string; even though the result is a Unicode string type, the characters are guaranteed to be valid ASCII suitable for use with non-Unicode libraries.
