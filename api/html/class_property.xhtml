const string & Property::LocalizedValue ( const std::locale &  locale) const

Obtains the value according to a given locale, if one is available.

Note that localized variants of a value can be specified by adding extensions to this property. The property's primary value may be matched to a locale by specifying a language value using SetLanguage(), but alternatives can only be specified using PropertyExtensions.

Parameters
localeA locale to use when searching for localized values.
Returns
A localized version of the property's value if available, or else returns the non-localized value as returned from the Value() method.
