Detailed Description

The Container class provides an interface for interacting with an EPUB container, i.e. a .epub file.

Each Container instance owns all its sub-items. All Packages from a single container, for instance, are kept around as pointers which will be deleted when the container is destroyed.

Remarks
The Container class holds owning references to the Archive instance used to read from the zip file, the XML document for the OCF file at META-INF/container.xml, all Packages within the container, and all EncryptionInfo instances from META-INF/encryption.xml.
