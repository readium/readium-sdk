Detailed Description

The RangeFilterContext abstract class is an extension of the FilterContext class, and it is used to pass data needed for extracting ranges of bytes from a given resource.

As you may imagine, one piece of data that we need to pass is the range of bytes that is being requested. That is passed by the ByteRange object that it is included in this class. In addition to that, the FilterContext object will need direct access to the ZIP file so that it can read only the pieces that it needs (after all, the whole idea was that we didn't want to cram a whole 1 GB video file in memory). That reference is kept the m_byteStream member variable.
