Member Enumeration Documentation

enum CFI::Component::Flags : uint8_t

Bitfield values identifying special status for a CFI component.

Enumerator
Qualifier 

The component contains an id or idref qualifier, i.e. [bob]

CharacterOffset 

The component has a character offset value, i.e. :12

TemporalOffset 

The component has a temporal offset value, i.e. ~87.24.

SpatialOffset 

The component has a spatial offset value, i.e. @150:220

Indirector 

The component specifies stepping into the identified node (i.e. !)

TextQualifier 

The component contains a text qualifier, i.e. [this]

SpatialTemporalOffset 

Component contains both temporal and spatial offsets.
